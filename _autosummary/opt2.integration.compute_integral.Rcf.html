opt2.integration.compute_integral.Rcf¶

Rcf(f, a, b, n)[source]¶

Compute numerical approximation using rectangle or mid-point method in an interval. Nodes are generated via formula: x_i = a+(i+1/2)h_hat for i=0,1,…,n-1 and h_hat=(b-a)/n :param f: function expression of integrand. :type f: function :param a: left point of interval. :type a: float :param b: right point of interval. :type b: float :param n: number of subintervals. :type n: float

Returns

numerical approximation to integral

of f in the interval a,b

Return type

sum_res (float)
